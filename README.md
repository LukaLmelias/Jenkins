# Jenkins docker container

# Usage:

### requirement:

- Install [docker](https://docs.docker.com/engine/install/)



### starting the contatiner
- clone this repo
  ```
  $ git clone https://github.com/LukaLmelias/Jenkins.git
  ```

- Enter into the Jenkins directory.

  ```
  $ cd Jenkins
  ```

- Run docker compose

  ```
  $ docker compose up 
  ```
